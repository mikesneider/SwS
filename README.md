# Shadow without showing - SwS
Este proyecto es para pruebas locales en donde probaré el enviar imagenes cifradas y luego descifrarlas para mostrar el resultado, sin mostrar la imagen original en ningún momento. Esto es para probar la seguridad de cifrado y descifrado de imágenes para el proyecto de aprendizaje federado utlizando la técnica de Shadow without showing (SwS).
